Tecnologias & Referências

Todas as tecnologias, ferramentas e referências utilizadas no desenvolvimento do IFRNChat.

// linguagens & padrões
HTML5
Linguagem de marcação para estruturação do conteúdo web.
CSS3
Estilização avançada com variáveis, flexbox, grid e animações.
JavaScript ES6+
Lógica de aplicação, autenticação e gerenciamento de estado.

Google Fonts
Fontes Syne e Space Mono para tipografia diferenciada.
Web Storage API
localStorage para persistência de sessão e mensagens do chat.
SVG
Gráficos vetoriais para ícones e elementos visuais inline.
